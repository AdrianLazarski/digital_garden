### Przykład

```csharp
int x = 2, y = 3; //deklaracja zmiennych
y = x--; //Najpierw do zmiennej y zostanie przypisana wartość x, potem x zostanie zmniejszone o 1
x = 1, y = 2 // Rezultat


int x = 2, y = 3; //deklaracja zmiennych
y = --x; //najpierw zmnięjszona zostanie wartość x, potem ta nowa wartość x zostanie przypisana do zmiennej y
x = 1, y = 1 //Rezultat
```
