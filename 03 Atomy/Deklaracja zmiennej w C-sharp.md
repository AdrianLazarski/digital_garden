```csharp
//typ_zmiennej nazwa_zmiennej;

//Przykład: 
int i;

//Podczas jednej deklaracji możemy zadeklarować wiele zmiennych tego
//samego typu:
−
int i, j, k;
−
int zmienna1, zmienna2, zmienna3;
−
int licznik, mianownik, wynik, wynik_dodawania;
```


> [!warning] Uwagi
> - Każda nazwa zmiennej musi być unikalna. 
> 
>- W C# istotne są małe i wielkie litery. 
>
>- Nazwa musi być jednym ciągiem znaków bez spacji; dozwolone jest użycie podkreślenia (np. `wynik_dodawania`).
