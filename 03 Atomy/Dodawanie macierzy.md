> [!note] Definicja
> **Sumą** macierzy $A$ i $B$ nazywamy macierz $C = [c_{ij}]$ której elementy określone są wzorem:  
> $$c_{ij} = a_{ij} + b_{ij}$$

> [!note] Definicja
> Różnicą macierzy $A$ i $B$ nazywamy macierz $C = [c_{ij}]$ której elementy określone są wzorem:  
> $$c_{ij} = a_{ij} - b_{ij}$$

![[Dodawanie macierzy.webp]]

> [!warning] Uwaga!
> Dodawać można tylko macierze o tych samych wymiarach, czyli takie, które należą do tego samego zbioru $M(m, n)$.

Przemienność dodawania
$A+B=B+A$
Łączność dodawania
$(A+B)+C = A+(B+C)$
Element neutralny dodawania
$A + 0 = 0 + A = A$
Element przeciwny
$A + (-A) = 0$
