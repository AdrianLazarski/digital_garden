- Jego składnia jest znajoma dla użytkowników języków C, C++, JavaScript lub Java.
- Język silnie typowany. Trzeba deklarować typ każdej zmiennej, zanim jej się użyje.
- Program napisany w języku C# składa się z instrukcji, każda z nich kończy się znakiem średnika (;)


```csharp

using System;

namespace MojaAplikacja // Przestrzeń nazwowa
{
    class Program //konwencjonalna nazwa dla klasy zawierającej metodę main
    {
        static void Main(string[] args) // Metoda 
        {
            Console.WriteLine("Witaj, Świecie!"); // Instrukcje programu
        }
    }
}
```
