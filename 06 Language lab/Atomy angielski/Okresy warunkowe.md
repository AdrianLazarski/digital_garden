## First Conditional

if + [[Present Simple]], will + bezokolicznik
will + bezokolicznik, if + [[Present Simple]]

- ***If** the weather is fine on Saturday, we'll go for a long walk.*
- *She will kiss you **if** you give her flowers.*


Używamy gdy jest bardzo duże prawdopodobieństwo, że warunek zostanie spełniony.


## Second Conditional

if + [[Past Simple]], would + bezokolicznik
would + bezokolicznik, if + [[Past Simple]]

- ***If** I had a lot of money, I would go on a trip around the world.*
- *She'd have more friends if she weren't so aloof.*

> [!warning] Uwaga!
> W drugim okresie warunkowym czasownik **be** użyty po `if` ma najczęściej formę `were`.
> 
>If I were you, I would never buy this CD.

Drugiego okresu warunkowego używamy mówiąc o teraźniejszości i przyszłości, jeśli istnieje **niewielkie prawdopodobieństwo** spełnienia warunku lub warunek **nie może** być spełniony.

## Third Conditional

if + [[Past Perfect]], [[Future Perfect]] in the Past
if + had + III forma, would have + III forma

- *We would have got married many years ago if I'd been more courageous.*
- *If she had known about the consequences, she wouldn't have done that.*

### Czasem zamiast Past Perfect pojawią się inne formy:

[[Past Perfect Continious]], jeśli czynność z przeszłości była tłem dla innej czynności

- *They wouldn't have caused that accident if they hadn't been driving so fast.*
- *I would have accepted his offer if I had been looking for a job.*

Pominięcie if oraz inwersja w języku formalnym:

- *Had taxes risen, there would have been strikes.*
- *Had the government reacted immediately, the riots wouldn't have continued.*

But for + rzeczownik/zaimek (oznacza to "Gdyby nie"):

- *But for his help, I wouldn't have passed the exam.*
- *But for them, we would have got lost.*

If it hadn't been for + rzeczownik/zaimek (oznacza to "Gdyby nie"):

- *If it hadn't been for their support, I wouldn't have finished university.*
- *Had it not been for that, we would have been in trouble.*

was/were + +have + III forma
Gdy chcemy podkreślić ograniczone prawdopodobieństwo wystąpienia jakiegoś wydarzenia w przeszłości

- If I was to have failed, my parents would have got furious.
- If we were to have gone bankrupt, we would have set up another company.

Zamiast would, może być could lub might.
