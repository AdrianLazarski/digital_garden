## 1. Ogólna konstrukcja

| Osoba       | To be | Skrót  | Verb + ing |
| ----------- | ----- | ------ | ---------- |
| I           | am    | I'm    | learning   |
| You         | are   | You're | learning   |
| He/She/It   | is    | He's   | learning   |
| We/You/They | are   | We're  | learning   |

### 1.1 Dodatkowe zasady

#### -ie na końcu wymieniamy na -y

| ie -> | y     |
| ----- | ----- |
| die   | dying |
| lie   | lying |
| tie   | tying |

#### -e na końcu wymieniamy na -i

| e ->  | ing     |
| ----- | ------- |
| make  | making  |
| take  | taking  |
| have  | having  |
| write | writing |
| dance | dancing |

#### Podwojenie spółgłoski 1
W czasownikach jednosylabowych, gdzie na końcu jest **spółgłoska + samogłoska + spółgłoska**, podwajamy ostatnią spółgłoskę, 
	przy czym podwajanie nie dotyczy spółgłosek w i x!

| litera | x2       |
| ------ | -------- |
| swim   | swimming |
| run    | running  |
| sit    | sitting  |
| stop   | stopping |
| get    | getting  |

#### Podwojenie spółgłoski 2
W czasownikach dwusylabowych, gdzie na końcu jest **spółgłoska + samogłoska + spółgłoska**, a akcent pada na drugą sylabę, podwajamy ostatnią spółgłoskę.

| litera | x2         |
| ------ | ---------- |
| refer  | referring  |
| begin  | beginning  |
| forget | forgetting |

## 2. Kiedy używać

### 1 Czynności dziejące się w momencie mówienia


|               |              |
| ------------- | ------------ |
| now           | teraz        |
| at the moment | w tej chwili |

### 2 Czynności trwające dłuższy czas (wraz z chwilą obecną, ale przekraczające ją)

Taka czynność ma charakter tymczasowy, nie jest niczym stałym. 

np. *I'm taking driving lessons.*

|            |                         |
| ---------- | ----------------------- |
| at present | obecnie                 |
| nowadays   | w obecnych czasach      |
| today      | dziś                    |
| tonight    | dziś wieczorem/w nocy   |
| these days | w tych dniach/aktualnie |
| this month | w tym miesiącu          |

### 3 Czynności zaplanowane

np. 
- *On Friday we are having an important meeting.*
- *Sorry, we are working this evening.*
- *Are you doing anything at the weekend?*
- *The President is having lunch with his advisory board and then he is meeting the managers.*
- *I can't see you on Tuesday afternoon. I'm seeing a customer.*

### 4 Opis czyjegoś zachowania, które nie jest stałą cechą

np. *My neighbours are always arguing so loud at night.*



## 3. Czasowniki nie występujące w Present Continious

|                                                                                                   |                                                                                                             |
| ------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| be  <br>believe  <br>want  <br>cost  <br>own  <br>know  <br>desire  <br>smell <br>taste <br>think | hear  <br>need  <br>see  <br>guess  <br>understand  <br>mean  <br>last  <br>belong  <br>prefer  <br>suppose |
smell w znaczeniu pachnieć, nie wąchać
taste w znaczeniu smakować, nie próbować, jak smakuje
think w znaczeniu uważać/mieć opinię, nie myśleć

Czasem są wyjątki. 
- np. w zdaniu *Why are you being naughty?* PC jest użyty w kontekście czwartym (Opis czyjegoś zachowania, które nie jest stałą cechą)
- Jednak w normalnej konstrukcji nie spytamy kogoś *Are you being en engineer?* tylko *Are you en enginner?*
