# Biografia 📁

- 1466-1536
- Filolog, filozof, pedagog, pisarz, duchowny. Czołowy humanista.
- Oskarżony o herezję, bo popierał konieczność zreformowania Kościoła, ale heretykiem nie był, proponowano mu nawet w swoim czacie kapelusz kardynalski.

# Poglądy 🔍
