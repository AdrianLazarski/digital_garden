
Schematy wnioskowania [[Indukcja|indukcyjnego]], wymyślone przez [[John Stuart Mill|Milla]].

Rozdział nazwał czterema metodami, a wymienia pięć. Może dlatego, że kanon zmian towarzyszących nie ma charakteru eliminacyjnego.


## Kanon jedynej zgodności (zawodne)

Jeżeli jakaś okoliczność stale towarzyszy występowaniu jakiegoś zjawiska, to jest ona jego przyczyną lub skutkiem.

| Sfera przyczyn | Sfera skutków |
| -------------- | ------------- |
| A, B, C, D, E  | F             |
| A, B, C, D     | F             |
| A, B, C, E     | F             |
| A, B, D, E     | F             |
| A, C, D, E     | F             |

Prawdopodobnie A jest przyczyną F (lub jej częścią).

  
## Kanon jedynej różnicy (zawodne)

Jeżeli jakaś okoliczność zachodzi gdy określone zjawisko występuje, a nie zachodzi gdy dane zjawisko nie występuje, a wszystkie pozostałe okoliczności są takie same, to okoliczność ta jest przyczyną lub skutkiem badanego zjawiska.

| Sfera przyczyn | Sfera skutków |
| -------------- | ------------- |
| A, B, C, D, E  | F             |
| A, B, C, D     |               |

Prawdopodobnie E jest przyczyną F (lub jej częścią).

  
## Połączony kanon jedynej różnicy i jedynej zgodności (zawodne)

W kanonie tym najpierw ustala się przy pomocy kanonu jedynej zgodności, czy określone zjawisko A jest przyczyną zjawiska D, a następnie stosuje się kanon jedynej różnicy w celu weryfikacji danego stwierdzenia. Większy stopień pewności.

  
## Kanon reszt (zawodne)

To co pozostaje niewyjaśnione w sferze skutkowej jest wynikiem oddziaływania tego, co pozostaje niewyjaśnione w sferze przyczynowej.

| Sfera przyczyn | Sfera skutków |
| -------------- | ------------- |
| A, B, C        | D, E          |
| A, B           | E             |

C jest przyczyną D. 


## Kanon zmian współtowarzyszących

Jeżeli jakieś zjawisko zmienia się wraz ze zmianą drugiego zjawiska, podczas gdy towarzyszące im okoliczności nie ulegają zmianie, to pomiędzy tymi zjawiskami zachodzi związek przyczynowy. Jedyny użyteczny w naukach empirycznych kanon, paradoksalnie uważany przez Milla za najmniej użyteczny. 

| Sfera przyczyn | Sfera skutków |
| -------------- | ------------- |
| A, B, C, D     | E             |
| zmA, B, C, D   | zmE           |
| A, zmB, C, D   | E             |
| A, B, zmC, D   | E             |
| A, C, zmD      | E             |

A jest przyczyną E.
