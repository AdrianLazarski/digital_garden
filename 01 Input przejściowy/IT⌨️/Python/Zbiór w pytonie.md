- Nieuporządkowany zestaw elementów.
- Elementy jednego zbioru mogą być różnych typów.
- Elementy się nie powtarzają. Są iterowalne i mutowalne.
- Elementy setu nie mają ustalonej kolejności i nie można ich indeksować ani sortować

Pusty zbiór
`s1 = set()`

s1 = set("GeeksForGeeks")


s2 = set(["Geeks", "For", "Geeks"])
