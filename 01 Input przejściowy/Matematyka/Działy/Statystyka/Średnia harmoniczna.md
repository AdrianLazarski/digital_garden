
Jest odwrotnością [[Średnia arytmetyczna|średniej arytmetycznej]] z odwrotności wartości zmiennych.

![[Pasted image 20240915151642.png]]
x<sub>i</sub> - wartości zmiennych
n<sub>i</sub> (i = 1, 2 .... k) - liczebność jednostek, odpowiadających poszczególnym wariantom zmiennej
N - ogólna liczebność badanej zbiorowości
N = n<sub>1</sub> + n<sub>2</sub> + ... + n<sub>k</sub>

- Drugi wzór dotyczy [[Szereg rozdzielczy|szeregów rozdzielczych]] punktowych.
- W szeregach rozdzielczych [[Szereg przedziałowy|przedziałowych]] x<sub>i</sub> jest zastąpione środkami przedziałów c<sub>i</sub>
	- c<sub>i</sub> = (x<sub>0i</sub>+x<sub>1i</sub>)/2  - środek przedziału klasowego
- W średniej harmonicznej z szeregów rozdzielczych liczebności są wagami.
