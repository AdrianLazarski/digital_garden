- Pierwsza to średnia nieważona.
- Średnia arytmetyczna z rozkładu rozdzielczego punktowego lub rozdzielczego przedziałowego jest ważona.

**Legenda do szeregu punktowego**
n<sub>i</sub> (i = 1, 2 .... k) - liczebność jednostek, odpowiadających poszczególnym wariantom zmiennej
N - ogólna liczebność badanej zbiorowości
N = n<sub>1</sub> + n<sub>2</sub> + ... + n<sub>k</sub>

**Legenda do [[Szereg przedziałowy|szeregu przedziałowego]]**
c¯ <sub>i</sub> = (x<sub>0i</sub>+x<sub>1i</sub>)/2  - środek przedziału klasowego
x<sub>0i</sub> - dolna granica przedziału
x<sub>1i</sub> - górna granica przedziału
  

Jeśli zamiast liczebności absolutnych w obliczeniach zostaną wykorzystane procentowe wskaźniki częstości, to wzór na średnia arytmetyczną szeregu przedziałowego przyjmuje postać:

# <span style="background:#d4b106">x = 1/N * (c<sub>1</sub>w<sub>1</sub> + c<sub>2</sub>w<sub>2</sub> + ... + c<sub>i</sub>w<sub>i</sub> )</span>

gdzie
# <span style="background:#d4b106">𝑤𝑖 = (𝑛𝑖/𝑁) ∗ 100</span>
# $w_i= \frac{{n_i}}{N} * 100$
